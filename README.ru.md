# AlexInd's Server Skins

Клиентский мод для Minecraft 1.7.10 Forge, заменяющий скины игроков на индивидуальные, получаемые через API.

## Как это работает

1. Для каждого игрока в мире отправляется GET-запрос на настроенный URL с ником игрока
2. Сервер возвращает JSON с ссылкой на скин
3. Мод скачивает PNG скина и применяет его к конкретному игроку через рефлексию

## Настройка

После первого запуска отредактируйте `config/alexinds_server_skins.cfg`:

```
general {
    S:apiUrl=http://localhost:8000/%s
}
```

`%s` заменяется на ник игрока.

## API

Мод отправляет GET-запрос на настроенный URL (где `%s` заменяется ником игрока).
Сервер должен вернуть JSON следующей структуры:

```json
{
    "SKIN": {
        "url": "https://example.com/path/to/skin.png",
        "metadata": {}
    }
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| `SKIN.url` | string | Прямая ссылка на PNG скина игрока (64x64 или 64x32) |
| `SKIN.metadata` | object | Зарезервировано, передавайте пустой объект `{}` |

Скин должен быть доступен по прямой ссылке (без редиректов и авторизации).

## Пример сервера

В папке `example-server/server.py` находится минимальный сервер на Python:

```
python example-server/server.py
```

Положите PNG скины 64x64 в `example-server/skins/` с именем `<ник>.png` (например, `Notch.png`).
Сервер сам отдаёт и JSON-ответ, и файлы скинов.

Настройка мода:
```
general {
    S:apiUrl=http://ваш-ip:8000/%s
}
```

## Сборка

```
./gradlew clean build
```

Результат: `build/libs/SkinReplacer-1.0.0.jar`
